Heron Language 3D Geometry Demo

Heron is a new statically typed functional programming language with a JavaScript-like syntax that specializes in the processing of arrays of numerical data.

This demo uses Heron to generate mesh data (vertices, faces, and colors) and uses Three.JS to display the resulting geometry. Browse the source code here:
