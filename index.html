Heron Language 3D Geometry Demo

Heron is a new statically typed functional programming language with a JavaScript-like syntax.

Heron is being developed for processing of arrays of numerical data and is intended to be easy to learn for non-programmers. The current implementation of Heron is written in TypeScript and generates JavaScript, but is designed so that it can easily target different platforms.

This demo uses Heron to generate mesh data (vertices, faces, and colors) for 3D meshes and uses Three.JS to display the resulting geometry.
